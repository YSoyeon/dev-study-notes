# 어댑터 패턴

> 호환되지 않는 인터페이스들을 연결하는 디자인 패턴

리액트 개발에서 어려운 점 중 하나는 백엔드 응답과 컴포넌트 데이터 구조가 일치하는지 확인하는 것이다.

어댑터 패턴은 특정 객체를 다른 객체의 구조에 맞게 '조정'할 수 있는 디자인 패턴이다.
